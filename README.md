# edc-consent
Add base classes for the Informed Consent form and process.

Installation
------------

	pip install edc-consent
	
Add to settings:

	MIN_AGE_OF_CONSENT = 16
	MAX_AGE_OF_CONSENT = 64
	AGE_IS_ADULT = 18
	GENDER_OF_CONSENT = ['M', 'F']
